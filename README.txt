# Мини-сайт для остатков крепежа

Файлы:
- index.html
- styles.css
- script.js
- products.js

Как использовать:
1. Откройте index.html в браузере.
2. Замените в блоке "Оставить заявку" ссылки `href="#"` на ссылку Авито, WhatsApp или Telegram.
3. Загрузите папку на хостинг или в Tilda Zero Block через HTML/T123.
4. Для обычного хостинга достаточно загрузить все 4 файла в одну папку.

Изображения категорий подключены через Unsplash Source по смысловым запросам.
